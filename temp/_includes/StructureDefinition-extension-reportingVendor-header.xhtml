http://hl7.org/fhir/us/davinci-deqm/StructureDefinition/extension-reportingVendor

A reference to the vendor who queried the data, calculated results and/or generated the report. The 'reporting vendor' is intended to represent the submitting entity when it is not the same as the reporting entity.
